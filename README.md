### Setup

Clone the repo with

```
  https://github.com/Berabjesus/math-magicians-calculator.git
```

Install packages with

```
  npm install
```

Start server with:

```
  npm start
```
Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
